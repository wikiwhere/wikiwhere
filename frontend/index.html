About

Wikiwhere is a tool meant to visualize relations between Wikipedia articles via links using directed graphs. It can both update relations on the fly and find the shortest path between any two articles.

Usage

  • Type in the first search box and click the search icon to visualize a single article and its relations/update the current graph.
  • Double click on any node to see that node's relations.
  • Type in both the first and second search boxes and click the search in the second box to search for a route between two articles (might take some time).
  • Hover over a node to see its descripton.
  • Click the clear button to clear the current visualization.
